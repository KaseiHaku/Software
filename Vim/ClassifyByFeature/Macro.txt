录制宏 {
    q           # 开启录制宏
    a           # 指定宏名称，其实就是保存 宏的寄存器名字，只能是 a-z 小写寄存器
    ...         # 一系列操作
    <ESC>       # 退回到 NormalMode
    q           # 结束宏 a 的录制
}

使用宏 {
    @a          # 执行宏 a 的一系列动作
    @@          # 重复执行上一次执行的宏
    3@a         # 执行 3 遍 宏 a
}

修改宏 {
    将录制的宏的寄存器直接粘贴到文本中编辑就行了，编辑好后复制回原本的寄存器
    qA          # 向 a 寄存器追加宏操作，这里是大写字母，大写字母表示追加
    
}
