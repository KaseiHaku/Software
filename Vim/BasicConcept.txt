" 查看命令的帮助文档
:h <name>

" 查看所有映射
:map

" 显示版本及默认 vimrc 配置文件加载路径
:version 

" 按加载顺序显示 vim 启动后所有 script 
:scriptnames 

" 查看加载的所有function
:function 

" 查看所有环境变量的设置
:set all
" 查看所有与系统默认不同的环境变量
:set
" 显示指定环境变量的当前值
:set variable?
" 显示script搜索路径
:set runtimepath?





" Vim 所有模式
Normal Mode                     nnoremap                        键入的字符是 命令, <ESC> 进入该模式
Operator Pending Mode           onoremap                        例如: Normal Mode 中按 d 光标会变成 半个, 这是就是这种模式 

Insert Mode                                                     i a o 进入该模式
Replace Mode
Insert and Replace Mode         inoremap

Visual Mode                     xnoremap                        键入的字符是 命令, v 进入该模式, V 进入行模式, Ctrl+V 进入块模式
Select Mode                     snoremap                        类似于 Visual Mode 但是键入的字符会替换 选区
Visual and Select Mode          vnoremap

Command-line Mode               cnoremap
Ex Mode                                                         多行 Command-line, Q 或 gQ 进入该模式, :vi 或 :q 退出该模式
