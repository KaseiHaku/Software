软件安装相关目录结构介绍：
    /usr                        # 系统级程序安装目录，所有系统级的软件都安装在该目录下
    /usr/lib                    # 系统级库文件目录，所有系统级软件依赖的库文件存放位置
    /usr/src                    # 系统级源码目录，存放源码

    /usr/local                  # 用户级程序安装目录，用于存放用户自己编译安装的软件
    /usr/local/src              # 用户级的源码目录

    /opt                        # 用户级的程序安装目录，安装到/opt目录下的程序，它所有的数据、库文件等等都是放在同个目录下面。不用时整个删除即可


软件包的名字含义介绍：
    zlib.x86_64                 # 表示是普通使用的包
    zlib-devel.x86_64           # -devel 表示：如果需要通过代码的方式，调用该软件包的中代码库，那么需要安装这个      

查看系统中已经安装的软件：
    shell> whereis mysql                       # 查看指定软件的安装路径，（适用源码安装）
    shell> which mysql                         # 查看运行文件所在路径，（适用源码安装）
